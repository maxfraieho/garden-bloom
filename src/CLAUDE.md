## API Invariant

**Frontend НІКОЛИ не звертається до бекенду напряму.**
Єдиний канал: `mcpGatewayClient` → Cloudflare Worker (gateway: `https://garden-mcp.exodus.pp.ua`).

Всі HTTP-запити через `requestJson<T>()`. Ніяких прямих `fetch()` до MCP backend. Виняток: статичні файли (`/graph.snapshot.json`, `/site/notes/...`) та зовнішні blob URLs.

Canonical API contracts: `docs/backend/КОНТРАКТИ_API_V1.md`
