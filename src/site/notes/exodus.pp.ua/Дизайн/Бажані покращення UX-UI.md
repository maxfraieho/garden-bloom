1. **Dashboard як «живий сад»** — візуалізація зон/агентів як органічна карта замість списку. Кожна зона — «грядка», кожен агент — «рослина» з індикатором активності.
    
2. **Proposal flow** — наочний flow схвалення пропозицій: картка пропозиції → diff-preview → approve/reject, з анімацією переходів.
    
3. **Unified editor** — єдиний редактор для нотаток, ДРАКОН-діаграм і конфігурацій агентів, з tabs і split-view.
    
4. **Status bar** — постійна нижня панель з статусом підключення до бекенду, останніми подіями агентів, кількістю pending proposals.
    
5. **Типографіка** — перехід на пару шрифтів: display (напр. Space Grotesk) + body (напр. IBM Plex Sans) для чіткої ієрархії.
    
6. **Кольорова палітра** — збагатити зелений акцент градієнтами (emerald → teal), додати теплий amber для warnings/proposals, зберегти dark base.
