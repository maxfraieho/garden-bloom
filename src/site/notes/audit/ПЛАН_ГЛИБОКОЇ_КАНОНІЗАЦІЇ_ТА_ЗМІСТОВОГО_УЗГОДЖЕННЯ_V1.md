## 2. Методологія

### Script executor fallback
Codex завершується з exit code 139 (SIGSEGV) при використанні Unicode символів в аргументах командного рядка (аналогічно до Phase 1). Phase 2 виконана через Python script executor відповідно до задокументованої fallback policy.

### Інваріанти, що зберігаються
- Технічні ідентифікатори (BLOOM, membridge, DiffMem, DRAKON, MinIO) — залишаються латиницею
- ENV vars, code blocks, CLI команди — без змін
- `workflow/` convention (lowercase-kebab-case) — не змінюється
- Англомовні виключення (PROJECT_DESCRIPTION_CANONICAL.md) — не змінюються
