### 1.1 API invariant — `requestJson<T>` / прямі `fetch()`

| Файл | Target | Природа |
|------|--------|---------|

| `src/hooks/useDrakonDiagram.ts:13` | `/site/notes/.../*.drakon.json` | Локальний статичний файл |
