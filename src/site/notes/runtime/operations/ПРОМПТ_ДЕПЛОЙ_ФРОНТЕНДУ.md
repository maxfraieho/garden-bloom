# Промпт для Claude на Alpine — деплой фронтенду

> Скопіюй у Claude CLI на Alpine сервері.

---

## Задача

Pull останні зміни з GitHub і зробити rebuild фронтенду для bloom.exodus.pp.ua.

### Кроки

```bash
cd /home/vokov/membridge/membridge

# 1. Pull
git pull origin main

# 2. Install deps (якщо змінився package.json)
npm install

# 3. Build
npm run build

# 4. Скопіювати білд у директорію nginx
# Визнач куди nginx роздає статику:
grep -r "root\|alias" /etc/nginx/conf.d/ /etc/nginx/sites-enabled/ /etc/nginx/nginx.conf 2>/dev/null | grep -i bloom

# Типово це щось як:
#   cp -r dist/* /var/www/bloom/
# або якщо nginx вказує прямо на dist/:
#   нічого копіювати не треба

# 5. Перевірка
curl -s https://bloom.exodus.pp.ua/nodes | head -20
```

### Перевірити після деплою

1. Відкрий https://bloom.exodus.pp.ua
2. Підключися до Runtime
3. Перейди на /nodes
4. Під кожним воркером має бути кнопка "Details" — натисни
5. Має з'явитися RAM, Disk, Uptime, Claude CLI info

Якщо `/system-info` endpoint ще не додано в агент — див. `docs/runtime/operations/ПРОМПТ_ПЕРЕВІРКА_HEALTH_CARDS.md`.
