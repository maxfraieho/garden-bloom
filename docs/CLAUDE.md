## Правила при роботі з документацією

1. **Не змінювати** Tier 1 canonical docs без явного запиту — вони є джерелом правди
2. **Нові docs** додавати з frontmatter (tags, created, updated, title, tier)
3. **Мова** нових docs — українська (якщо не зазначено English canonical)
4. **Кожен новий doc** має бути пов'язаний з _INDEX.md відповідного каталогу
5. **Historical docs** позначати `status:historical` і **не оновлювати**
6. **Orphan docs** (без посилань на них) — заборонені. Завжди додавати до _INDEX.md
