# Модель домену — Frontend словник сутностей

> **Для кого:** Lovable (JSX naming), Claude (типи і хуки), будь-який розробник що додає нові компоненти.
>
> Ціль: уніфіковані імена сутностей у JSX, TypeScript, і API запитах.

---

## Ключові сутності

### Note (Нотатка)

Основна одиниця контенту в системі.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `slug` | `string` | URL-ідентифікатор (`/notes/:slug`) |
| `title` | `string` | Заголовок |
| `content` | `string` | Markdown/Wikilinks вміст |
| `tags` | `string[]` | Теги для навігації |
| `backlinks` | `NoteRef[]` | Вхідні посилання з інших нотаток |
| `created_at` | `string` | ISO 8601 |
| `updated_at` | `string` | ISO 8601 |

**UI naming:** `NoteCard`, `NoteList`, `NotePage`, `useNoteEditor`

**API naming:** `GET /notes/:slug`, `POST /notes`, `PATCH /notes/:slug`

---

### Agent (Агент)

AI-агент зареєстрований у системі.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `id` | `string` | UUID |
| `name` | `string` | Відображувана назва |
| `type` | `string` | Тип агента |
| `status` | `AgentStatus` | `active` / `idle` / `error` |
| `capabilities` | `string[]` | Що агент вміє |

**UI naming:** `AgentCard`, `AgentForm`, `AIAgentBadge`, `AgentsPage`

**Hook:** `useAgentRegistry`

---

### Run (Виконання)

Один запуск агента з початком, кінцем і статусом.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `run_id` | `string` | UUID |
| `agent_id` | `string` | Який агент |
| `status` | `RunStatus` | `pending` / `running` / `completed` / `failed` |
| `started_at` | `string` | ISO 8601 |
| `finished_at` | `string \| null` | ISO 8601 або null якщо ще йде |
| `artifacts` | `Artifact[]` | Артефакти що виникли під час запуску |

**UI routes:** `/runtime/execution`, `/runtime/history`, `/runtime/artifacts`

**Pages:** `ExecutionPage`, `HistoryPage`, `ArtifactsPage`

---

### Proposal (Пропозиція)

Дія, яку агент **пропонує** — людина **погоджує або відхиляє**.

> Це центральна аксіома системи: AI proposes, human decides.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `proposal_id` | `string` | UUID |
| `run_id` | `string` | Зв'язаний запуск |
| `action` | `string` | Що агент хоче зробити |
| `status` | `ProposalStatus` | `pending` / `approved` / `rejected` / `applied` |
| `payload` | `object` | Дані для виконання дії |

**State machine:** [[../../operations/СИСТЕМА_PROPOSAL_V1]]

---

### Zone (Зона)

Обмежена область доступу з власним токеном і терміном дії.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `zone_id` | `string` | UUID |
| `name` | `string` | Назва зони |
| `token` | `string` | Токен доступу |
| `expires_at` | `string \| null` | Термін дії або null (безстроково) |
| `access_type` | `'web' \| 'mcp' \| 'both'` | Тип доступу |

**UI routes:** `/zone/:zoneId`, `/zone/:zoneId/edit/:noteSlug`, `/admin/zones`

**Pages:** `ZoneViewPage`, `ZoneEditPage`, `AdminZonesPage`

**Hook:** `useAccessZones`, `useZoneValidation`

**Error:** `ZONE_EXPIRED` / `ZONE_NOT_FOUND` → AccessGateUI відображає access wall

---

### Memory (Пам'ять агента)

Довгострокова пам'ять агента у форматі git diff / temporal revisions.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `memory_id` | `string` | UUID |
| `agent_id` | `string` | Чия пам'ять |
| `depth` | `MemoryDepth` | `present` / `history` / `diff` |
| `content` | `string` | Markdown або diff |
| `revision` | `number` | Номер ревізії |
| `created_at` | `string` | ISO 8601 |

**UI components:** `DepthSwitcher`, `HistoryPanel`, `DiffViewer`, `RevisionSnapshotViewer`

**Специфікація:** [[../diffmem/_INDEX]]

---

### Artifact (Артефакт)

Файл або вивід що виник в результаті виконання агента.

| Поле | Тип | Призначення |
|------|-----|-------------|
| `artifact_id` | `string` | UUID |
| `run_id` | `string` | Зв'язаний запуск |
| `type` | `ArtifactType` | `file` / `log` / `json` / ... |
| `path` | `string` | Шлях або URL |
| `size_bytes` | `number` | Розмір |

**UI route:** `/runtime/artifacts`

---

### Tag (Тег)

Мітка нотатки для навігації та групування.

**UI routes:** `/tags`, `/tags/:tag`

**Pages:** `TagsIndex`, `TagPage`

**Hook:** `useTags`

## Gateway Error Codes

Помилки через `GatewayErrorCode` у `types/mcpGateway.ts`:

| Код | Сценарій |
|-----|----------|
| `AUTH_REQUIRED` | Немає owner токену |
| `ZONE_EXPIRED` | Зона закінчилася |
| `ZONE_NOT_FOUND` | Зону видалено |
| `FORBIDDEN` | Немає доступу до ресурсу |
| `INVALID_TRANSITION` | Неможлива зміна статусу (proposal/run) |
| `AGENT_TIMEOUT` | Агент не відповів |
