# Створення React компонента

Допоможи створити новий компонент для проекту.

## Інформація для збору:

1. **Назва та призначення**
   - Як називається компонент?
   - Що він робить?
   - Де буде використовуватись?

2. **Пропси**
   - Які пропси потрібні?
   - Які типи цих пропсів?
   - Які пропси обов'язкові, які опційні?

3. **UI компоненти**
   - Які shadcn-ui компоненти використати?
   - Чи потрібно додати нові shadcn-ui компоненти?
   - Якою має бути структура компонента?

4. **Стан та логіка**
   - Чи потрібен локальний стан?
   - Чи потрібні хуки?
   - Чи потрібна валідація?
   - Чи потрібні API запити (React Query)?

## Структура створення:

1. Перевірити чи існують схожі компоненти
2. Створити TypeScript інтерфейс для пропсів
3. Створити базову структуру компонента
4. Додати стилі (Tailwind)
5. Додати логіку та хуки
6. Експортувати компонент
7. Створити приклад використання
