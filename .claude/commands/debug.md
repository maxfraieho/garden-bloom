# Дебаг проблеми

Допоможи знайти та виправити проблему в проекті.

## Систематичний підхід до дебагу:

1. **Збір інформації**
   - Яка саме помилка? (повний текст помилки)
   - Коли виникає? (при якій дії)
   - Де виникає? (який файл, компонент)
   - Чи є stack trace?

2. **Аналіз помилки**
   - TypeScript помилка?
   - Runtime помилка в браузері?
   - Build помилка Vite?
   - Помилка в стилях?
   - Помилка в логіці компонента?

3. **Пошук причини**
   - Перевірити типи TypeScript
   - Перевірити імпорти
   - Перевірити пропси компонентів
   - Перевірити стан (React Query, useState)
   - Перевірити хуки (правила хуків)
   - Перевірити роутинг

4. **Тестування гіпотез**
   - Створити мінімальний репродуктивний приклад
   - Ізолювати проблему
   - Перевірити одну зміну за раз

5. **Виправлення**
   - Зробити мінімальне виправлення
   - Перевірити що помилка зникла
   - Перевірити що нічого не зламалось
   - Запустити `npm run build` для перевірки TypeScript
   - Запустити `npm run lint`

**Використовуй явні DOING/EXPECT/RESULT блоки для кожної гіпотези!**
