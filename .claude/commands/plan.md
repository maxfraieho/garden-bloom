# План нової фічі

Допоможи мені розпланувати нову функціональність для цього React + Vite проекту.

## Що потрібно зробити:

1. **Аналіз вимог**
   - Зрозуміти що саме потрібно реалізувати
   - Визначити які компоненти вже існують і можна переиспользувати
   - Визначити які нові компоненти потрібно створити

2. **Технічний план**
   - Які файли потрібно створити/змінити
   - Які shadcn-ui компоненти використати
   - Яку структуру даних використати
   - Чи потрібна валідація форм (Zod схеми)
   - Чи потрібен state management (React Query)
   - Чи потрібні нові роути

3. **Порядок виконання**
   - Розбити на логічні кроки
   - Визначити залежності між кроками
   - Створити чіткий план імплементації

4. **Питання**
   - Які є невизначеності?
   - Які є альтернативні підходи?
   - Що потрібно уточнити у користувача?
