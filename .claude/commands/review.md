# Code Review

Перевір зміни перед комітом.

## Чеклист для перевірки:

### TypeScript
- [ ] Всі типи правильно вказані
- [ ] Немає `any` (або є вагома причина)
- [ ] Інтерфейси для всіх пропсів компонентів
- [ ] `npm run build` проходить без помилок

### React
- [ ] Компоненти правильно структуровані
- [ ] Хуки використані згідно з Rules of Hooks
- [ ] Пропси деструктуровані
- [ ] Немає memory leaks (useEffect cleanup)
- [ ] Key props для списків

### Стилі
- [ ] Використано Tailwind класи
- [ ] Використано `cn()` для умовних класів
- [ ] Немає inline стилів без причини
- [ ] Responsive design де потрібно

### shadcn-ui
- [ ] Використані існуючі компоненти
- [ ] Нові компоненти додані через CLI
- [ ] Компоненти правильно кастомізовані

### Форми та валідація
- [ ] React Hook Form налаштований
- [ ] Zod схеми створені
- [ ] Error messages зрозумілі
- [ ] Валідація працює

### State Management
- [ ] React Query для server state
- [ ] Query keys правильні
- [ ] Error handling присутній
- [ ] Loading states оброблені

### Якість коду
- [ ] Немає console.log (або є причина)
- [ ] Змінні та функції мають зрозумілі назви
- [ ] Код не дублюється
- [ ] `npm run lint` проходить

### Тестування
- [ ] Перевірено в браузері
- [ ] Всі інтерактивні елементи працюють
- [ ] Немає помилок в консолі
- [ ] Responsive layout працює

**Після перевірки запусти `npm run build && npm run lint` і тільки тоді коміть!**
